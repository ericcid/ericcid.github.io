Cantilever Truss

This project involved designing, constructing, and testing a cantilever half-truss panel out of aluminum members in order to withstand a load of 1000 pounds. The objective was to obtain the highest truss score, which was calculated by analyzing the collected data and determining which truss minimized the amount of sag, or displacement, and weight. The forces of tension and compression were calculated using Bilige Yang's Truss program. Multiple designs of the truss were developed in order to determine the best one int terms of ratio of vertical stiffness to actual weight, factor of safety with respect to maximum member loads, and minimization of joint twisting moment with certain given restrictions. The truss was tested by applying a load with the Universal Testing Machine (UTM). The data on sag at various points and total weight were collected and analyzed using MATLAB.

SwatRoDI

The purpose of this project was to build a bot that could be programmed wirelessly, via WiFi and HTTP, and could receive multiple instructions to control an LED, a buzzer, and motors. The use of HTTP to control SwatRoDI (the bot) allows users to program it using any technology or programming language that supports HTTP requests. Support for this is widely available nowadays in modern programming languages such as Python, Java, and C++. However, as part of the project, we also developed a user-friendly interface to allow non-technical users to program SwatRoDI more easily. The name of the project was inspired by a similar existing project called RoDI, which has its own customized Arduino board.
